# Try this

- **backend**: https://codeleap-network-app.fly.dev/posts.json
- **frontend**: https://crud-django-react-3sdq.vercel.app

# How to run

## Backend

### Dependencies

- python3

```Bash
  cd backend && pip install && python manage.py runserver
```

## Frontend

- nodejs 18
- pnpm

```Bash
  cd frontend && pnpm install && pnpm dev
```
